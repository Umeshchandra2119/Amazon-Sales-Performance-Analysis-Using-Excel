## üß† **Key Insights & Findings**

* **Top Product:** Laptop / Electronics products contributed highest sales.
* **Best Performing Category:** Electronics and Fashion.
* **Top City:** Major metro cities recorded the highest order volume.
* **Order Status Breakdown:**
  * 31% Cancelled
  * 35% Completed
  * 34% Pending
* **Payment Method Trend:** Majority orders were prepaid using UPI/card.
* **Revenue Drivers:** High-value electronics contributed maximum revenue.
